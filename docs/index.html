About the project

Goodbirds pulls recent Notable sightings from eBird for selected radii and renders them to folium Leaflet maps. Each city page links to the current build for each city. The goal is fast loading, clear labeling, and a clean path to original eBird checklists.

Built with: eBird API, Python, Requests, folium on top of Leaflet, GitHub Pages, and GitHub Actions.
